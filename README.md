# Echoes of the Crime

A murder-mystery game built in Unity where each case — suspects, clues, and
branching investigation outcomes — is generated by a fine-tuned LLM instead
of being hand-written, so no two playthroughs present the same mystery.

## How it works

1. The **Loading** scene requests a new case from a hosted LLM endpoint
   (`LLMStoryClient`, configured via `LLMEndpointConfig`) and parses the
   response into suspects, clues, and starting actions (`StoryParser`).
2. `GameSession` carries that generated case into the **Game** scene.
3. The player investigates by choosing actions; each choice is sent back to
   the LLM for a contextual result and, eventually, a guess at the culprit.

See [`docs/llm-training`](docs/llm-training) for the notebook used to
fine-tune the model that powers case generation.

## Tech

- Unity, C#
- TextMeshPro for UI
- A LoRA-fine-tuned LLM served from a hosted inference endpoint

## Running locally

Open the project in Unity Hub and press Play from the `MainMenu` scene.
An LLM endpoint must be configured (`Assets/Config/LLMEndpointConfig.asset`,
not tracked in git since it holds an API key) for case generation to work.
